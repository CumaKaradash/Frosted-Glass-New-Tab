# Frosted Focus - New Tab

**Frosted Focus**, Chrome'un varsayılan "Yeni Sekme" sayfasını estetik, minimalist ve odaklanmayı artıran "buzlu cam" (frosted glass) efektli bir arayüzle değiştiren bir eklentidir.

##  Özellikler

- **Minimalist Tasarım:** Dikkat dağıtıcı unsurlardan arındırılmış temiz arayüz.
- **Modern UI:** Modern web teknolojileri ve CSS efektleri ile görsel şölen.
- **Hafif Yapı:** Gereksiz scriptler barındırmaz, hızlı yüklenir.

##  Geliştirme ve Kurulum

Bu proje modern frontend araçları ile geliştirilmiştir (Vite/React yapısı görünmektedir).

1. Projeyi indirin.
2. `chrome://extensions/` adresinden klasörü yükleyin.
3. Yeni bir sekme açtığınızda tasarımın değiştiğini göreceksiniz.

##  Yapı

- `manifest.json`: New Tab override ayarları.
- `index.html`: Ana giriş noktası.
- `assets/`: Derlenmiş JS ve CSS dosyaları.
